# MeteoNow
Découvrez les conditions météorologiques actuelles de n'importe quelle ville avec MeteoNow, l'application météo moderne et élégante. Grâce à une interface intuitive, accédez en un instant à :  - La température actuelle - La sensation thermique (ressenti) - Le taux d'humidité - Les conditions météorologiques détaillées  Conçue avec Vue.js
